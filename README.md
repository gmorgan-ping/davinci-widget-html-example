# PingOrchestrate Sample Flow Embed

This glitch application allows you to embed your PingOrchestrate widget within an HTML page.

## Usage

Set the following variables within the index.html page (lines 18-21)

- skApiBaseUrl
- companyId
- skApiKey
- policyId
